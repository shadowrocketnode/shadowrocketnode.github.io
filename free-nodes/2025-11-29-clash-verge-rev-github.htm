逐步指导：添加订阅的全过程

现在让我们进入核心环节——如何在Shadowrocket中添加订阅。请确保你已经安装了最新版本的Shadowrocket，然后按照以下步骤操作：

第一步：启动应用并进入设置界面 打开Shadowrocket后，你会看到一个简洁但功能丰富的主界面。右下角的"设置"图标是你的控制中心，点击进入后你会发现多个选项。这里需要特别注意的是，首次使用时应检查网络权限设置，确保应用拥有完整的网络访问权限。

第二步：导航至订阅管理 在设置菜单中，"订阅"选项通常位于比较显眼的位置。点击进入后，你会看到"添加订阅"的按钮。这个界面还包含了更新频率设置、自动更新开关等重要选项。建议将更新频率设置为每天自动更新，以确保始终使用最新的节点信息。

第三步：填写订阅信息 点击"添加订阅"后，会出现一个信息填写表单。这里需要输入两个关键信息：备注名称和订阅链接。备注名称建议使用容易识别的名称，比如服务商名称+套餐类型；订阅链接则粘贴之前获取的完整URL。高级用户还可以在这里设置代理规则和策略组，但对于大多数用户来说，使用默认设置即可。

第四步：保存并更新 填写完成后点击右上角的"保存"按钮，系统会自动开始首次订阅更新。这个过程可能需要几秒钟到一分钟不等，取决于节点数量的多少。更新完成后，你会主界面的服务器列表中看到所有可用的节点。

FAQ

SSR 的端口复用有什么风险？
SSR 支持多端口配置以复用服务，但若端口过多或设置冲突，可能被扫描工具识别或误判为异常服务。建议保持配置简洁，并合理分配端口范围以减少暴露风险。
如何在 Clash for Android 中导入订阅链接？
Clash for Android 支持通过订阅链接快速导入节点。用户只需在配置界面中点击“新建配置”，选择“URL 导入”并粘贴订阅地址即可，程序会自动下载并解析所有节点配置。
WinXray 的 PAC 模式适合什么场景？
PAC 模式通过 JavaScript 文件动态判断哪些流量走代理，哪些直连。适合访问国内外混合网站环境，用户无需手动切换节点即可根据规则自动分流。
Nekobox 是否支持自定义 DNS 解析？
支持。用户可以在配置中指定 DNS 服务器或启用 DoH/DoT 功能，确保域名解析安全和快速，避免本地 DNS 污染或被劫持。
QuantumultX 如何通过策略组优化多应用分流？
QuantumultX 可以根据应用类型、端口和协议设置策略组，自动选择最优节点处理不同流量。这样可保证网页浏览、视频播放和游戏访问顺畅，提高整体网络效率。
Sing-box 如何按端口分流不同流量？
Sing-box 支持按端口匹配流量，将不同应用分配到指定节点。结合策略组使用，可实现精准分流，提高访问速度和网络稳定性，优化整体网络体验。
VMess 使用 TLS+WebSocket 的好处是什么？
TLS+WebSocket 可伪装成正常 HTTPS 流量，增强抗封锁能力和隐蔽性。结合域名伪装和策略组使用，可在高封锁网络环境下保持安全、稳定和高速的网络访问体验。
WinXray 多策略组如何管理大量节点？
多策略组可将节点按地区、延迟或用途分类，并结合规则文件自动分流流量。这样可快速选择最优节点，提高访问速度和稳定性，同时优化节点管理和维护效率。
如何检查 Clash 连接失败的日志以定位问题？
打开 Clash 的日志面板或查看配置目录下的 log 文件，观察报错信息如 DNS 解析失败、节点拒绝连接或 TLS 握手错误。根据关键字（例如 timeout、refused、certificate）逐项排查并比对服务器配置。
Nekoray 如何实现节点延迟自动切换？
Nekoray 支持配置策略组（Policy Group），用户可设置 URL 测试或周期检测机制，根据延迟自动选择最优节点。此功能能提升稳定性，尤其适合多线路机场订阅用户。
