第三章：手把手安装配置教程

3.1 前期准备：获取必要信息

就像组装乐高需要图纸，配置V2Ray需要这些"零件"：
- 服务器地址（通常是域名）
- 端口号（警惕非常用端口如10086可能是陷阱）
- UUID（相当于你的专属加密钥匙）
- 传输协议和伪装类型

3.2 Shadowrocket配置实战（图文详解）

  1. 基础设置：

    • 点击右上角"+"选择Vmess协议
    • 填入服务器信息时，注意区分"address"和"host"的区别

  2. 传输协议选择：

    • WebSocket + TLS：适合企业网络
    • mKCP + 视频伪装：应对校园网限制

  3. 高级技巧：

    • 开启"TFO"加速（TCP Fast Open）
    • 设置"Fallback"备用节点

常见踩坑：忘记在「DNS设置」中关闭「本地DNS映射」会导致国内网站访问异常。

第四章：高阶玩家进阶手册

4.1 规则配置的艺术

通过编辑config.json文件，你可以实现：
json "routing": { "domainStrategy": "IPIfNonMatch", "rules": [ { "type": "field", "outboundTag": "direct", "domain": ["geosite:cn"] } ] }
这段代码意味着所有中国网站直连，其他走代理——这就是智能分流的核心逻辑。

  • 多路复用：在弱网环境下提升30%速度
  • 动态端口：每10分钟更换端口躲避检测
  • 负载均衡：多个服务器智能切换

第五章：安全使用指南

5.1 服务商选择避坑原则

  • 警惕"永久免费"陷阱
  • 优先选择支持Alipay/Stripe支付的商家
  • 查看IP是否被Netflix等流媒体封禁

5.2 隐私保护三重奏

  1. 定期更换UUID（建议每月一次）
  2. 启用TLS1.3加密
  3. 关闭客户端的日志记录功能
